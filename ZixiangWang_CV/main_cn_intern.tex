{面向交通仿真实验结果分析，开发自然语言驱动的报告智能体，降低实验分析门槛}
{
\begin{itemize}
    \item 负责基于 FastAPI 和 FastMCP 开发报告智能体服务，对接交通仿真实验平台 HTTP API，封装仿真配置创建、结果获取、日志拉取等能力，为自然语言驱动的实验分析提供统一入口。
    \item 基于 LangGraph 设计智能工作流，串联需求解析、配置校验、仿真执行、日志分析、报告生成与可视化创建等节点，并补充会话状态、重试与历史管理机制。
    \item 实现仿真分析模块，从日志中重建订单、车辆与乘客状态，计算匹配率、完成率、等待时间、行程时间、绕行指标与车辆利用率等统计结果，自动生成分析报告和图表。
    \item 将静态工具升级为 MCP + LangGraph 的智能服务形态，支持自然语言请求、批量仿真分析和参数优化，增强实验平台的可用性与自动化程度。
\end{itemize}
}
